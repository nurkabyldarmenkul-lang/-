# Student Management System

## 📌 Description
This project is a **Student Management System** developed to manage student data efficiently.  
It allows users to add, update, delete, and view student information.

## 🚀 Features
- Add new students
- View student list
- Update student details
- Delete students
- Search students

## 🛠️ Technologies Used
- Java
- MySQL
- Git & GitHub
